3.1 Simple implementation

Description:

Simply wrap each slide to be shown with an element (e.g. div) an give it a class named "slide". All slides need now to be wrapped with another HTML element with a class of your choice. Your code may look like this: <div class="slideshow">
  <div class="slide">
    <img src="images/1.jpg"/>
  </div>
  <div class="slide">
    <img src="images/2.jpg"/>
  </div>
  <div class="slide">
    <img src="images/3.jpg"/>
  </div>
</div>
 Now all you need is the following snippet to be implemented in your HTML-document: <script type="text/javascript">
  $(document).ready(function () {
    $('.slideshow').squeezeGallery();
  });
</script>

Options

The following options are provided.
Name Default Description
slidesMaxRatio 0.618 Ratio of the expanded slide width to gallery width. Values from 0.1 to 0.9 are available.
sliderEasing swing Defines the easing method for slides. You may want to you use the jQuery easing plugin.
duration 500 Defines the duration of the slides-animation.
teaser null Define an element by class to animate some extra teaser text.
galleryWidth null Define a custom width for the gallery container, otherwise it will as big as its parent element.
galleryHeight null Define a custom height for the gallery, otherwise height will be defined by the smallest slide available.
